Cascading Stitle Sheets

Definição

Cascading Stytle Sheets - Flhas de Estilo em Cascata - tem por objetivo realizar a estilização de conteúdo de documentos HTMl. Podem definir cores, fontes, espaçamento, alinhamnetos, bordas dentre outros.

Componentes

  • Seletores - seleciona elementos HTML com base no nome do elemento ao qual o estilo será aplicado
  • Propriedades - o aspecto específico do elemento que quer usar como estilo
  • Valores - a configurações aplicada ao atributo

Formas de utilização

Inline

Os códigos CSS são definidos dentro do uma tag HTML.

Incorporado ou Internal

Os códigos CSS são definidos no cabeçalho, dentro da tag head. Afetam somente o documento em que foi definido.

Externo ou External

Os códigos CSS são definidos em um arquivo externo. Afetam todos os decumentos que estão linkados ao CSS.
